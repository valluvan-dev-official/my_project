============= Jinja Tags =============


1. Variable Tag : {{ }}

    Python la irukura value ah HTML la Display panna use aaguthu



2. If Condition :  {% if condition %}  {% else %}  {% endif %}

    Condition check panna 


3. For Loop : {% for %} {% endfor %}

    list/dictionary  iterate panna


4. Template Inheritance(extends / block) :

    Re-usable Template Structure


5. Include Tag : {% include %} 

    small HTML piece(partial ) add panna



6. Comment :

    {# #} - Jinja Comment 



7. Filter :  | filtername

    Modify Variable Values


8. Static files - {% static %}

    load css ,js or images


9. URL Tag - {% url %}


10. Now - {% now %}

    show current data/time


11. regroup :

    {% regroup list by field as new_variable %}
